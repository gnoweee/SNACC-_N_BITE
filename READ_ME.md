# SNACC 'N BITE Ordering Program

This is a simple **C++ console program** that simulates an ordering system for a snack and fast-food store called **SNACC 'N BITE**.

---

## Features

- Display main menu with food items and add-ons
- Take orders with quantity input
- Option to edit or remove items before finalizing
- Calculate subtotal, tax (12%), and discount (20% if total ≥ ₱500)
- Print a formatted receipt
- Handle payment and calculate change

---

## How to Run

1. Download or clone the repository.
2. Open `main.cpp` in a C++ IDE or text editor.
3. Compile and run the program using your preferred compiler, e.g.:

```bash
g++ main.cpp -o snacc
./snacc   # on Linux/Mac
snacc.exe # on Windows
